# Lahman Baseball Database
This project is for developing Data Visualizations and Dashboards using the Lahman Baseball Database

## Getting Started
To replicate this Project: 
* Download and install [SQL Server Developer Edition](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) and [SQL Server Management Studio](https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-2017) 
* Import Files: [Batting](https://github.com/CentsOfFate/Lahman-Baseball-Database/blob/master/Files/Batting.xlsx), [People](https://github.com/CentsOfFate/Lahman-Baseball-Database/blob/master/Files/People.csv), [Salaries](https://github.com/CentsOfFate/Lahman-Baseball-Database/blob/master/Files/Salaries.xlsx), [Teams](https://github.com/CentsOfFate/Lahman-Baseball-Database/blob/master/Files/Teams.xlsx)
* Create View using [vLahman](https://github.com/CentsOfFate/Lahman-Baseball-Database/blob/master/Files/Create%20vLahman.sql)
